ClojureBridge Helsinki

6.4.2019

Futurice, Annankatu 34B, 8. kerros, 00100 Helsinki .

The next ClojureBridge event in Helsinki is organized on Saturday 6.4.2019. The event takes place at the Futurice office in Annankatu 34B, 8. floor, 00100 Helsinki .

ClojureBridge aims to increase diversity within the Clojure community by offering free, beginner-friendly Clojure programming workshops for underrepresented groups in tech.

ClojureBridge is free and targeted to people who identify themselves as women.

The event is organized on 6.4.2019 at 9-17. We start by installing all the needed software together.

The requirement is that every participant has their own laptop that has wifi and admin rights to install software. If you are using Windows, please make sure that your operating system is at least version 7.

This event will be run either in English or Finnish depending on the preferences of the audience. Please tell us which language you'd prefer when you sign up.

The preliminary agenda for the day

  • Introduction: How does a computer work, what is programming?
  • Installation and basic usage of the tools
  • Coffee break
  • Programming
  • Lunch
  • More programming
  • Coffee break
  • Even more programming
  • Summary, links and hints: How to continue with software development?
  • The official program ends at 17 o'clock.

Sivu suomeksi

Everyone who identify themselves as women are welcome in the event. If there are more people signed up than we can host, we will draw lots on Monday 25.3.2019 at 12:00 to select participants randomly.

If there are free spots, also men are welcome as avecs.
